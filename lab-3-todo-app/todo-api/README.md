# TODO App - REST API 

## Build

### Yarn

```bash
yarn install
```

## Run

### Yarn

Development:

```bash
yarn serve
```

Production:

```bash
yarn start
```


## Linting

To run ESLint, execute:

```bash
yarn lint
```

## Documentation

Once you run the application, the documentation of the API can be found at [http://localhost:8080/swagger](http://localhost:8080/swagger)
